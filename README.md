# Netflix Originals Analysis

This repository contains a Python project for analyzing a dataset of Netflix original films. The analysis includes data preprocessing, exploratory data analysis (EDA), visualizations, and insights on film releases, IMDB scores, runtimes, and top-rated films.

## Dataset

The dataset used is `netflix_originals.csv`, which contains information about Netflix original films including:

- Title  
- Premiere date  
- Runtime  
- IMDB score  

## Features

- **Yearly Trends:** Line chart showing the number of Netflix original films released per year.  
- **IMDB Score Distribution:** Histogram of IMDB scores across all films.  
- **Runtime vs IMDB Score:** Scatter plot analyzing the relationship between movie runtime and IMDB rating, with a trend line.  
- **Top 10 Movies Table:** Tabular visualization of the top 10 movies based on IMDB scores.

## Installation

1. Clone the repository: 
`git clone https://github.com/yourusername/netflix-originals-analysis.git`
2. Install required Python packages:
`pip install pandas numpy matplotlib'

## Usage

1. Update the csv_path variable in the script with the path to your CSV file.
2. Run the Python script:
`python netflix_analysis.py`
3. The script will generate plots and a table showing insights from the dataset.
